🔥 Setup Guide
Step 1 — Create Firebase Project

Go to console.firebase.google.com → Add Project → name it edutrack → disable Google Analytics → Create.

Step 2 — Create Realtime Database

In your project → Build → Realtime Database → Create Database → choose your closest region → Start in test mode → Enable.

Step 3 — Get your Database URL

After creating the database, copy the URL shown at the top — it looks like:
https://your-project-default-rtdb.firebaseio.com

Step 4 — Get your Web API Key

Project Settings (⚙️) → General → Your apps → click </> Web → Register app → copy the apiKey value.

Step 5 — Paste below and Connect

Paste the two values on the right, click Connect & Sync. Share the same HTML file with your teachers — everyone will see the same live data.

🔑 Firebase Config

Upload your existing data to Firebase once to share with all devices.

📥 Bake Config into File (Required for GitHub)

This downloads a new version of the app with your Firebase config permanently embedded. Upload this file to GitHub — then ALL devices (public dashboard, other teachers) will connect to Firebase automatically without any setup.
